IUSHPay
Sistema unificado de pagos y control de acceso para el parqueadero de la Institución Universitaria Salazar y Herrera.

Descripción
IUSHPay es una plataforma digital que integra una wallet institucional, validación mediante QR dinámico y conexión con pasarelas de pago (PSE), permitiendo a estudiantes y visitantes gestionar pagos y accesos en tiempo real.

El sistema elimina inconsistencias de saldo, reduce filas en portería y mejora la trazabilidad de las transacciones mediante una arquitectura segura, eficiente y escalable.

Problemática
El sistema actual presenta:

Pagos que no se reflejan en tiempo real

Inconsistencias de saldo

Validaciones manuales en portería (filas largas)

Riesgo de fraude con carnet/cédula

Baja trazabilidad y auditoría

Esto afecta atributos clave como seguridad, fiabilidad, rendimiento y usabilidad.
